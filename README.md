# Learn With Claude

A personal learning experience powered by Claude Code.

No courses. No videos. No syllabus to follow.
Just you, Claude, and whatever you want to learn.

---

## What you need

- A [Claude Code](https://claude.ai/code) subscription
- This folder open in Claude Code

That's it. No setup. No installs. No configuration.

---

## How to start

1. Open this folder in Claude Code
2. Type `/start`
3. Follow the conversation

Claude will ask you a few questions, build a course outline tailored to you, and start teaching. At your pace.

---

## What can you learn?

Anything, but this works especially well for:

- Programming languages (Python, JavaScript, Java…)
- Tools and technologies (Docker, Git, SQL, APIs…)
- Concepts (system design, data structures, how the web works…)

If you're not sure what to learn, just say so. Claude will help you figure it out.

---

## What happens to your progress?

Everything is saved locally in the `learner/` folder — your profile, your course outline, where you left off. Next time you open Claude Code, type `/start` and it picks up right where you stopped.

---

## Who is this for?

Anyone. You don't need to be a developer. You don't need prior experience.
All you need is curiosity and a Claude Code subscription.
